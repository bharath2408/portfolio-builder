# Create a Portfolio

Creating a portfolio is the first step toward showcasing your work online. Foliocraft provides templates to jumpstart your design and a blank canvas for full creative freedom.

## Starting a New Portfolio

1. Navigate to **Portfolios** from the sidebar
2. Click the **+ New Portfolio** button
3. Choose a **template** to start with pre-built sections and blocks, or select **Blank** to start fresh
4. Enter a **title** for your portfolio — this becomes the heading displayed in the browser tab
5. A **slug** is generated automatically from your title (e.g., "My Portfolio" becomes `my-portfolio`). You can customize this
6. Click **Create**
7. You'll be taken to the Studio Editor where you can start building

## Template Selection

Templates come with pre-configured sections, blocks, and theme settings so you can start editing content immediately instead of building from scratch.

### Available Templates

- **Developer Portfolio** — includes Hero section with heading and CTA button, About section with stats, Skills section with skill bars/grid, Projects section with project cards, and Contact section with form
- **Designer Portfolio** — visual-heavy layout emphasizing images, galleries, and project showcases
- **Minimal** — clean, text-focused layout for writers, consultants, or anyone who prefers simplicity
- **Blank** — completely empty canvas — just a single empty section to start with

> **Tip:** Choosing a template doesn't lock you in. Every block, section, color, font, and layout can be modified, added, or removed in the editor. Templates are starting points, not constraints.

### What Templates Include

Each template comes with:
- **Pre-built sections** with names like Hero, About, Skills, Projects, Contact
- **Pre-populated blocks** with placeholder content ready to be replaced with your own
- **Theme settings** — coordinated colors, fonts, and styling
- **Layout configurations** — sections set up with appropriate layouts (flex, grid, absolute)

## Portfolio Settings

After creating a portfolio, you can update its settings from the editor's top bar (click the gear icon):

- **Title** — the name shown in the browser tab, dashboard, and navigation
- **Slug** — the URL path component. Only lowercase letters, numbers, and hyphens. Each slug must be unique across your portfolios
- **Description** — a brief summary used for SEO meta description
- **Status** — Draft (only visible to you) or Published (accessible via URL)

## Managing Your Portfolios

On the **Portfolios** page in your dashboard, you can perform these actions on any portfolio:

### Edit
Opens the portfolio in the Studio Editor where you can modify all sections, blocks, styles, and themes.

### Preview
Opens a full-screen live preview showing exactly how visitors will see your published portfolio. Test all links, buttons, and animations.

### Duplicate
Creates an exact copy of the portfolio with all sections, blocks, and styles. The copy gets a new slug with a number suffix (e.g., `my-portfolio-2`). Useful for creating variations or A/B testing designs.

### View Analytics
Opens the analytics page showing view counts, device breakdowns, referrer sources, and visitor timeline for published portfolios.

### View Submissions
Shows all contact form submissions received through the portfolio's contact form block.

### Copy URL
Copies the portfolio's public URL to your clipboard for sharing.

### Delete
Permanently removes the portfolio and all its data — sections, blocks, themes, analytics, and submissions. A confirmation dialog appears to prevent accidental deletion. You must confirm by clicking "Delete" in the dialog.

> **Warning:** Deletion is irreversible. Consider exporting your data first if you might need it later.

## Portfolio URLs

Your portfolio URLs follow this pattern:

- **Specific portfolio:** `yourdomain.com/portfolio/username/slug`
- **Default portfolio:** `yourdomain.com/portfolio/username`

Set a default portfolio in **Settings > Preferences > Default Portfolio** to make it accessible at your username URL directly.
