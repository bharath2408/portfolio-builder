# Your Dashboard

The dashboard is your home base in Foliocraft. It gives you an overview of your portfolios, quick access to all features, and a comfortable workspace for managing your work.

## Overview Page

When you log in, you'll see the **Overview** page displaying:

- **Portfolio count** — the total number of portfolios you've created
- **Total views** — combined view count across all published portfolios
- **Recent activity** — a feed of your latest portfolio updates, publishes, and edits

## Sidebar Navigation

The sidebar on the left is always visible on desktop and accessible via the hamburger menu on mobile.

### Main Section
- **Overview** — the dashboard home with stats and recent activity
- **Portfolios** — list, create, edit, duplicate, and manage all your portfolios

### Account Section
- **Settings** — your profile, appearance preferences, password, notifications, default portfolio, data export, and account deletion

### + New Portfolio Button
At the top of the sidebar, a prominent button to quickly start creating a new portfolio.

## Top Navigation Bar

The horizontal bar at the top of every dashboard page includes:

- **Breadcrumbs** — shows your current location (e.g., Overview > Portfolios > My Portfolio)
- **Search bar** — quickly search your portfolios. Keyboard shortcut: press `/` to focus the search
- **Notification bell** — alerts for portfolio views, form submissions, and system updates. A teal dot indicates unread notifications
- **Theme toggle** — sun/moon icon to switch between light and dark mode for the dashboard. The change applies immediately and persists across sessions
- **Sign out button** — log out of your account
- **User avatar** — your profile picture (from Cloudinary upload or OAuth provider) or initials if no picture is set

## Portfolios Page

Navigate to **Portfolios** in the sidebar to see all your portfolios displayed as cards.

### Portfolio Card Information
Each card shows:
- **Title** — the portfolio name
- **Status badge** — Draft (gray) or Published (green)
- **Last updated** — relative time (e.g., "2 hours ago")
- **View count** — number of visits (published portfolios only)

### Portfolio Card Actions

Hover over a card to see action buttons:

- **Edit** (pencil icon) — opens the Studio Editor
- **Preview** (eye icon) — opens a live preview of the portfolio
- **Analytics** (chart icon) — view visitor statistics
- **Duplicate** (copy icon) — create an exact copy with a new slug
- **Open live** (external link icon) — open the published portfolio URL in a new tab
- **Copy URL** (clipboard icon) — copy the portfolio URL to your clipboard
- **Delete** (trash icon) — permanently delete with a confirmation dialog

### Creating a New Portfolio

1. Click **+ New Portfolio** in the sidebar, or the create button on the Portfolios page
2. Select a template or choose **Blank** to start from scratch
3. Enter a title for your portfolio
4. Click **Create** — you'll be taken to the Studio Editor

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `/` | Focus the search bar |
| Theme icon click | Toggle light/dark mode |

## Responsive Design

The dashboard is fully responsive:
- **Desktop** — sidebar is always visible, content area is wide
- **Tablet** — sidebar collapses, accessible via hamburger menu
- **Mobile** — optimized layout with stacked cards, touch-friendly buttons
