# Welcome to Foliocraft

Foliocraft is a modern portfolio builder that lets you create stunning, professional portfolio websites — without writing any code. Whether you're a developer, designer, freelancer, or student, Foliocraft gives you everything you need to showcase your work beautifully.

## What You Can Do

- **Build visually** — Use the drag-and-drop studio editor with 30+ block types including headings, images, project cards, skill bars, shapes, groups, custom SVGs, and icons. A **command palette** (`Ctrl/Cmd + K`) gives instant access to 40+ actions
- **Embed rich content** — Add **integration blocks** for YouTube videos, Spotify playlists, Google Maps, Calendly scheduling, GitHub contribution graphs, and custom HTML embeds
- **Customize everything** — Adjust colors with the **advanced color picker** (gradients, eyedropper, theme tokens, swatches), fonts, spacing, shadows, animations, and hover effects with pixel-perfect control. **Copy and paste styles** between blocks to keep your design consistent
- **Design per-device layouts** — Style blocks differently for desktop, tablet, and mobile with **per-device style overrides** that save automatically when switching device previews
- **Work with precision** — **Smart alignment guides** snap blocks into place, **canvas rulers** show exact measurements, and **frame drag and resize** give you full control over section layout
- **Group and organize** — **Block grouping** (`Ctrl+G`) lets you treat multiple blocks as a single unit. Use **rubber-band marquee selection** to select several blocks at once
- **Use decorative elements** — Browse **shape presets** (dividers, blobs, geometric), pick from the **icon picker**, or **import custom SVGs** to add unique visual flair
- **Add background patterns** — Sections support **background patterns** for subtle texture and depth
- **Save portably** — Export and import your portfolio as a **.folio file** — a portable format you can share, back up, or transfer between accounts
- **Publish instantly** — Go live with one click and share your portfolio URL with the world
- **Track performance** — See who's visiting your portfolio with built-in analytics. A **notification bell** delivers live alerts for contact form submissions
- **Stay in control** — Version history lets you preview and restore any previous state. **Trash and soft-delete** give you 30-day recovery for deleted portfolios. Export all your data anytime
- **Generate a resume** — Turn your portfolio data into a polished **PDF resume** with a two-column layout and colored sidebar
